----------------------> Project Installation <--------------------------

1. install xampp into your computer and start xampp control.

2. open : http://localhost/phpmyadmin/ ---> into your browser.

3. create new database and give name "whatsappmenu".

4. import whatsappmenu.sql file into your database.

5. now, copy project folder to xampp->htdocs->whatsappmenu 

6. new open in browser : http://localhost/whatsappmenu and it will run.
